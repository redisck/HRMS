# HRMS（人力资源管理系统）OOAD-系统分析与技能训练
===========================
系统主要分为部门管理、职员管理、培训管理、奖惩管理、 薪资管理、报表管理、系统管理七大管理模块，从各个方面记录、整合企业人员的信息。
